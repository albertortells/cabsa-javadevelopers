# cabsa-javadevelopers

Prueba técnica para el proceso de selección de desarrollador Java en CABSA.

## Enunciado

Se propone desarrollar una pequeña API que permita:

- Devolver un listado de animales en JSON y lo que comen.
- Buscar animales por nombre o por lo que comen.
- Crear un nuevo tipo de comida.

Se parte de un esqueleto de aplicación en Spring Boot ya arrancable, con el objetivo de que la prueba no lleve más de una hora u hora y media.

> Se valorará más la estructura de código y la limpieza del mismo que la funcionalidad en sí.

Repositorio base facilitado: https://github.com/testcabsa/javadevelopers
